📌 기본 기능 구현 목표 

- Movie
    - 관리자 페이지에서 영화 등록 / 수정 / 삭제
    - 영화 관련 글 작성 가능한 커뮤니티
        - 영화 상세 페이지에서 한줄평 & 별점 등록 / 수정 / 삭제
        - 리뷰 / 건의 / 자유 게시판에서 영화에 대한 자유로운 게시글 작성 / 수정 / 삭제
            - 로그인한 사용자만 가능
    - 영화 추천 알고리즘
    - 게시글 & 댓글 수정시각 및 생성시각 표시
        - 사용자 친화적으로 구현하기 위해 댓글은 방금전, 1분전 등으로 표시 + 게시글은 시각을 `YYYY - mm -dd yy:mm` 형식으로 표현
